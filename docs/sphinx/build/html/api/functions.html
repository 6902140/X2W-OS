size_t memset(void *dst, byte value, size_t size)

memset将det_起始的size个字节的内存单元的值设置为value

参数:

  • dst – 起始地址

  • value – 设置的值

  • size – 要设置多少个字节

返回:

size_t 成功设置的字节数

size_t memcpy(void *dst, const void *src, size_t size)

memcpy将src起始的size个内存单元的值(以字节计算)复制到dst起始的size个内存单元中

警告

memcpy不会进行长度检查, 注意src和dst的长度, 以避免溢出

参数:

  • dst – 被复制的内存区域的起始地址

  • src – 复制到的内存区域的起始地址

  • size – 复制的字节数

返回:

size_t 复制的字节数

int8_t memcmp(const void *a, const void *b, size_t size)

memcmp用于比较两个分别以a和b为起始地址的的size个字节的内存区域的值

参数:

  • a – 要比较的第一个内存区域的地址

  • b – 要比较的第二个内存区域的地址

  • size – 要比较的字节数

返回:

int8_t 比较的结果:

  • 若a>b, 则返回1

  • 若a=b, 则返回0

  • 若a<b, 则返回-1

char *strcpy(char *dst, const char *src)

strcpy复制src指向的以’\0’结尾的字符串到dst指向的字符串去中去

警告

strcpy不会进行长度检查, 注意src和dst的长度, 以避免溢出

参数:

  • dst – 将要复制到的内存地址

  • src – 被复制的字符串

返回:

char* 返回dst的首地址

size_t strlen(const char *str)

strlen返回以’\0’结尾的字符串的长度

参数:

str – 要获取长度的字符串

返回:

size_t 字符串的长度

int8_t strcmp(const char *a, const char *b)

strcmp比较两个字符串

参数:

  • a – 要比较的第一个字符串

  • b – 要比较的第二个字符串

返回:

int8_t 返回比较的结果

  • 若a>b, 则返回1

  • 若a=b, 则返回0

  • 若a<b, 则返回-1

char *strcat(char *dst, const char *src)

strcat用于将字符串src拼接到dst后

警告

strcat不会进行长度检查, 注意src和dst的长度, 以避免溢出

参数:

  • dst – 将被拼接到的字符串

  • src – 将被拼接的字符串

返回:

char* 返回dst的首地址

char *strchr(const char *str, const uint8_t ch)

strchr返回字符串从左往右第一次出现字符ch的地址

备注

strchr相当于返回字符ch第一次出现的地址

参数:

  • str – 被搜索的字符串

  • ch – 要搜索的字符

返回:

char* 指向字符的指针,

  • 若字符ch在字符串str中, 则返回字符串第一个ch的地址

  • 若字符ch不在字符串str中, 则返回 NULL

char *strrchr(const char *str, const uint8_t ch)

strrchr返回反向字符串第一次出现字符ch的地址

备注

strrchr相当于返回字符ch最后一次出现的地址

参数:

  • str – 被搜索的字符串

  • ch – 要搜索的字符

返回:

char*

  • 若字符ch在字符串str中, 则返回反向字符串第一个ch的地址

  • 若字符ch不在字符串str中, 则返回 NULL

size_t strchrs(const char *str, const uint8_t ch)

  • str – 要被搜索的字符串

  • ch – 要查询的字符

size_t 字符ch在字符串str中出现的次数
