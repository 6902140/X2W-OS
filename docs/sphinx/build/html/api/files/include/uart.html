uart.h#

uart.h提供了包括初始化、输出字符串等函数在内的uart功能

Author

Shihong Wang (jack4shihong@gamil.com)

Version

0.1

Date

2023-04-08

Copyright

Copyright Shihong Wang (c) 2023 with GNU Public License V3.0

备注

QEMU默认模拟的UART设备处理芯片是AXI UART 16550

备注

关于AXI UART 16550芯片介绍: https://wenku.baidu.com/view/6f7a176048d7c1c708a14521.html?_wkts_=1681021571718

备注

关于AXI UART 16550相关常量, 参考AXI UART 16550编程手册(Programming Table): http://byterunner.com/16550.html

警告

目前使用轮询方式使用UART设备, 没有实现中断方式

警告

uart.h仅提供UART相关功能, AXI UART 16550相关常量的定义在asm/uart.h中

Functions

void uart_init(void)#

uart_init初始化UART设备.

备注

uart_init将UART设备的波特率设置为constrains.h中UART_BAUD_RATE所定义的值

备注

QEMU默认模拟的UART处理芯片为AXI UART 16550芯片

char uart_get(void)#

uart_get从UART设备接受一个字符

备注

目前uart_get是轮询的方式获得字符的, 未来在完成中断后需要改为中断式获得字符

void uart_put(char chr)#

uart_put向UART设备发送一个字符

警告

目前以轮询方式发送

参数:

chr – 要发送的字符

void uart_puts(const char *string)#

uart_puts向UART设备发送字符串

备注

uart_puts调用uart_put发送

参数:

string – 要发送的字符串
