constrains.h#

constrain.h定义了内核的一系列常数

Author

Shihong Wang (jack4shihong@gmail.com)

Version

0.1

Date

2023-04-09

Defines

DEBUG#

内核是否处于DEBUG模式, 若:

  • DEBUG = 0, 不会运行测试函数

  • DEBUG = 1, 运行测试函数

WITH_FP_REG#

CPU是否具有浮点寄存器, 若

  • WITH_FP_REG = 0, CPU 不具有浮点寄存器

  • WITH_FP_REG = 1, CPU 具有浮点寄存器

KERNEL_JUMP_ADDR#

内核的跳转地址, 在kernel.ld中定义

PRINTF_STRING_SIZE#

kprintf和uprintf可以输出的字符串最大长度

UART_BAUD_RATE#

UART设备的波特率

MAX_TEST_FUNCTION_NUM#

最大测试函数的数量

MAX_INTR_EXCP_INFO_NUM#

中断/异常处理函数信息最大数量
