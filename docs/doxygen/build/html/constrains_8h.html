宏定义说明

◆ DEBUG

#define DEBUG   1

内核是否处于DEBUG模式, 若:

  • DEBUG = 0, 不会运行测试函数
  • DEBUG = 1, 运行测试函数

◆ WITH_FP_REG

#define WITH_FP_REG   0

CPU是否具有浮点寄存器, 若

  • WITH_FP_REG = 0, CPU 不具有浮点寄存器
  • WITH_FP_REG = 1, CPU 具有浮点寄存器
