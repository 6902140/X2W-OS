◆ ktrap_enter()

void ktrap_enter ( void  )

ktrap_enter是内核的陷入入口函数, 由汇编实现, 定义在 ktrap_entry.S中

注解
该函数会:
  1. 在S模式的栈中构建被打断程序的陷入帧
  2. 将被打断的程序的陷入帧作为参数调用ktrap_dispatcher函数
