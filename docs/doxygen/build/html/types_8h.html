宏定义说明

◆ DEPRECATED

#define DEPRECATED   __attribute__((deprecated))

DEPRECATED宏用于通知GCC/Clang编译器被修饰的对象已经弃用, 帮助编译器静态分析用

注解
NO_RETURN宏本质上是借助了GCC/Clang的__attribute__关键词实现的, 使用的标记(annotation)是: deprecated
参考: https://renenyffenegger.ch/notes/development/languages/C-C-plus-plus/GCC/__attribute__/deprecated/index

◆ NO_RETURN

#define NO_RETURN   __attribute__((noreturn))

NO_RETURN宏用于通知GCC/Clang编译器被修饰的函数不会返回, 帮助编译器静态分析用

注解
NO_RETURN宏本质上是借助了GCC/Clang的__attribute__关键词实现的, 使用的标记(annotation)是: noreturn
参考: https://stackoverflow.com/questions/25408795/tell-gcc-that-a-function-call-will-not-return

◆ UNREACHABLE

#define UNREACHABLE   __builtin_unreachable()

UNREACHABLE宏用于通知GCC/Clang编译器不会运行到此处, 帮助编译器静态分析用

注解
NO_RETURN宏本质上是借助了GCC/Clang内建的__builtin_unreachable()函数实现的
参考: https://stackoverflow.com/questions/25408795/tell-gcc-that-a-function-call-will-not-return

◆ UNUSED

#define UNUSED   __attribute__((unused))

UNUSED宏用于通知GCC/Clang寄存器被修饰的变量可能没有被使用, 从而在编译的时候静默警告

注解
UNUSED宏本质上是借助了GCC/Clang的__attribute__关键词实现的, 使用的标记(annotation)是: unused
参考: https://gcc.gnu.org/onlinedocs/gcc-3.4.6/gcc/Variable-Attributes.html
