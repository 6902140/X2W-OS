详细描述

gtrapframe_t是通用陷入帧(General Trap Frame), 主要用于陷入中断/异常保存通用寄存器, 以方便在退出中断/异常的时候恢复现场

注解
  1. gtrapframe_t是否包含浮点寄存器取决于constrain.h的WITH_FP_REG宏:
    • WITH_FP_REG = 0, 不包含浮点寄存器, 此时保存: x1~x31 寄存器. 一共 31 个寄存器, 共 248 字节
    • WITH_FP_REG = 1, 包含浮点寄存器, 此时保存: x1~x31, f0~f31 一共 31 + 32 个寄存器, 共 248 + 128 字节
  2. 寄存器命名规则参考: RISC-V Assembly Programmer’s Handbook
警告
目前不支持浮点寄存器, 支持浮点寄存器的话, 需要额外修改trap_entry.S里的汇编入口函数来保存
