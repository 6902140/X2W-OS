{\ttfamily test\+\_\+func\+\_\+pointer\+\_\+t}测试函数指针类型 

\begin{DoxyNote}{注解}
测试函数的通用原型应该是{\ttfamily int test\+\_\+\+X\+X\+X\+X(void)}, 例如\+:
\begin{DoxyItemize}
\item {\ttfamily uart}的测试函数{\ttfamily int \mbox{\hyperlink{test__uart_8h_abbdb4b9ab6b89d7a6f6dfeb7e52e6c91}{test\+\_\+uart(void)}}} 
\end{DoxyItemize}

函数名即为函数的地址/函数第一条指令的地址, 因此函数名就是一个函数指针, 例如\+:
\begin{DoxyItemize}
\item 函数原型是{\ttfamily int \mbox{\hyperlink{test__uart_8h_abbdb4b9ab6b89d7a6f6dfeb7e52e6c91}{test\+\_\+uart(void)}}}, 则{\ttfamily test\+\_\+uart}就是指向{\ttfamily int (void)}的函数指针 
\end{DoxyItemize}

函数类型由函数的参数和返回值决定, 因此不同参数返回值的函数的函数类型不同, 因此不能放到数组中. 但是函数指针都是一个指针, 因此可以放到数组中, 所以这里声明的是函数指针 
\end{DoxyNote}
